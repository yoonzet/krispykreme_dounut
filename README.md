## 크리스피크림도넛 리뉴얼 (https://yoonzet.github.io/krispykreme_dounut/)

Html, scss, jQuery 기반의 리뉴얼 웹사이트 입니다.
반응형웹으로 제작되었습니다.

### 사용 기술

- SCSS를 이용한 마크업
- 미디어쿼리를 이용한 반응형 UI 마크업
- key frame을 이용한 동적인 이미지표현
- 라이브러리 활용(JQuery)
  - full page
  - slick slide
  - scroll magic
